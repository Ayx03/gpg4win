About gpg4win

The gpg4win project aims to provide and maintain a Windows-Installer for GnuPG and related products (GPA, WinPT, GPGol, Sylpheed-Claws, GPGee, EurdoraGPG as well as documentation).

It is an international project. Apart from the english as default, currently german language is fully supported, maintainers for other languages are welcome.

The main difference compared to similar projects (GnuPP and Windows Privacy Tools) is that at first we developed the gpg4win-Builder. This one allows to very easily create new gpg4win.exe installers with updated components. This could even be done on a GNU/Linux system where almost all products are automatically cross-compiled.

With this approach we hope to avoid quick aginf of installer packages as it happened in the past with the other approaches, because it is quite simple to create updates.

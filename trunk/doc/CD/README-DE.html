Inhalt der Gpg4win CD Edition

Gpg4win ist ein Installationspaket für Windows (2000/XP/2003/Vista) mit Computer-Programmen und Handbüchern für EMail- und Datei-Verschlüsselung.

Werfen Sie als erstes einen Blick in "Gpg4win für Einsteiger", wenn Sie Gpg4win noch nicht kennen und es nun installieren und nutzen möchten.
